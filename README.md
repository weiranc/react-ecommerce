## Installation

1. Clone the repo

   ```sh
   git clone https://github.com/your_username/react-ecommerce.git
   ```
2. Install NPM packages

   ```sh
   npm install
   ```
3. Update `.env` file if needed
4. Run webpack

   ```sh
   npm start
   ```
5. Connect to server

   ```sh
   npm run server
